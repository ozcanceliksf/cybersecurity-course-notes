# Cybersecurity Course Notes
**CTS2106 — Information Security Fundamentals | Santa Fe College**

Structured notes and summaries from cybersecurity coursework. Covers threats, risk management, security frameworks, and defensive concepts.

---

## Topics Covered

| Topic | Description |
|-------|-------------|
| Security Fundamentals | CIA Triad, authentication, authorization |
| Threat Landscape | Malware, phishing, social engineering, ransomware |
| Risk Management | Risk assessment, mitigation strategies, BCP |
| Network Security | Firewalls, IDS/IPS, VPNs, DMZ |
| Cryptography | Symmetric/asymmetric encryption, hashing, PKI |
| Ethical Hacking | Reconnaissance, scanning, exploitation basics |
| Incident Response | Detection, containment, eradication, recovery |
